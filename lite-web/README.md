# Lite Web Version

`lite-web`은 설치 없이 브라우저에서 파일명을 생성해 복사/내보내기만 하는 경량 버전입니다.

## 포함 기능

- 강의자료 모드 / 필족 모드 전환
- 시작 시 기본 1개 행 자동 생성
- `행 추가`로 제목 행 계속 생성
- 과목/순서 선택 시 시간표 기준으로 날짜/교시/시수/교수/강의제목 자동 채움
- 필족 모드에서 시수 `1` 고정
- 마지막으로 선택한 과목 기억 (`localStorage`)
- 최종 파일명 검토/수정
- 선택 항목 파일명 복사
- CSV 내보내기

## 제한 사항

- 브라우저 보안상 실제 파일 이름 변경/저장은 하지 않습니다.
- 생성된 파일명 텍스트를 복사하거나 CSV로 내려받아 사용합니다.

## 실행 (일반 사용자용)

1. `lite-web` 폴더를 통째로 전달
2. 상대방이 `index.html` 더블클릭
3. 브라우저에서 바로 사용

즉, `npm`이나 터미널 명령이 필요 없습니다.
